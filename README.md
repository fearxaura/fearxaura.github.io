# FEARLESS | Professional Developer Portfolio

A modern, black & white developer portfolio built with Vite 5, React 18, and TailwindCSS v3.

## Tech Stack

- **Framework**: Vite 5 + React 18
- **Styling**: TailwindCSS v3.4
- **Animations**: Framer Motion v11
- **Forms**: React Hook Form
- **Icons**: Lucide React
- **PWA**: vite-plugin-pwa + Workbox

## Sections

- **Hero** — Particle canvas animation + typewriter effect
- **About** — Animated skill bars with IntersectionObserver
- **Services** — Service cards (Web, Bots, VPS, Performance)
- **Portfolio** — Filterable project grid
- **Testimonials** — Auto-scrolling carousel
- **Contact** — Validated contact form + social links
- **Footer** — Social links

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
